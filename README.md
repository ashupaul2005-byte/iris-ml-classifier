# Iris ML Classifier

This project trains a Machine Learning model on the Iris dataset using Python and Scikit-Learn.

## Features
- Train-Test Split
- Decision Tree / Random Forest Model
- Cross Validation
- Classification Report
- Model Saving & Loading

## How to Run
pip install -r requirements.txt
python iris_model.py


## Output
Displays:
- Accuracy
- Cross Validation Scores
- Classification Report
